title: Dashboards
description: Build custom monitoring dashboards
---

Dashboards let you combine multiple topologies and metric widgets into a single view.

## Creating a Dashboard

1. Navigate to **Dashboards** in the sidebar
2. Click **Add Dashboard**
3. Enter a name and save

## Adding Widgets

Click the **+** button on the dashboard to add widgets. Available widget types:

- **Topology** — Embedded topology viewer with live metrics
- **Device Status** — Donut chart showing device status breakdown
- **Alerts** — Active alerts list from connected data sources

## Layout

Widgets can be resized and rearranged by dragging. The layout is saved automatically.

## Sharing

Click the **Share** button on a dashboard to generate a public link. Shared dashboards display live data without requiring authentication.
