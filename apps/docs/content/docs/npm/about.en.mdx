## What is Shumoku

Shumoku is an open-source library for **defining network diagrams in YAML/JSON and outputting them as SVG/HTML/PNG**.

It aims to help network engineers easily create and manage network diagrams within documentation.

## Features

- **Text-based** — Define networks in YAML. Easy version control and diff review with Git
- **Auto layout** — ELK.js-based hierarchical layout engine
- **Vendor icons** — 900+ icons including Yamaha, Aruba, AWS, Juniper
- **Interactive** — HTML output with pan/zoom, tooltips, hierarchical navigation
- **System integration** — Auto-generation from NetBox or custom APIs

## Origin of the Name

"Shumoku" (撞木) is a T-shaped stick used to strike bells in Japanese. The name was chosen because network topology diagrams resemble the shape of a shumoku.
