## Quick Start

### 1. Define your network in YAML

```yaml title="network.yaml"
name: "My Network"

nodes:
  - id: router
    label: "Core Router"
    type: router

  - id: switch
    label: "Main Switch"
    type: switch

  - id: server
    label: "Web Server"
    type: server

links:
  - from: router
    to: switch
    bandwidth: 10G

  - from: switch
    to: server
    bandwidth: 1G
```

### 2. Render with CLI

```bash
# SVG output
npx shumoku render network.yaml -o diagram.svg

# Interactive HTML output
npx shumoku render network.yaml -f html -o diagram.html

# PNG output
npx shumoku render network.yaml -f png -o diagram.png
```

### 3. Use vendor icons

```yaml
nodes:
  - id: router
    label: "RTX3510"
    type: router
    vendor: yamaha
    model: rtx3510
```
